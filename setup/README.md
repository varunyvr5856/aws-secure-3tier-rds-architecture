# AWS Console Setup

This folder documents how the infrastructure was created manually using
the AWS Management Console.

## Setup Order
1. Custom VPC creation
2. Public and private subnet configuration
3. Internet Gateway and route tables
4. Bastion Host EC2 setup
5. PostgreSQL Client EC2 setup
6. Amazon RDS PostgreSQL setup
7. Security group configuration and validation

## Key Console Sections Used
- VPC Dashboard
- EC2 Dashboard
- RDS Dashboard
- Security Groups
